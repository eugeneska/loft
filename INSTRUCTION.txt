
ИНСТРУКЦИЯ ПО РЕДАКТИРОВАНИЮ ЗАЛОВ


1. КАК РЕДАКТИРОВАТЬ ЗАЛЫ В HTML (ВЕРСТКА СТРАНИЦЫ ЗАЛА)


Файлы залов находятся в корне проекта:
- armaloft.html - зал Армалофт
- merkuri.html - зал Меркури
- samolet.html - зал Самолет
- rufer.html - зал Руфер
- pulka.html - зал Пулька

Для каждого зала можно редактировать следующие элементы:


 НАЗВАНИЕ ЗАЛА

Строка 1213: <h1 class="room-detail-title">АРМАЛОФТ</h1>
            ↑ Здесь меняйте название зала (например, "МЕРКУРИ", "САМОЛЕТ" и т.д.)



 ФОТОГРАФИИ В ГАЛЕРЕЕ                                                        

Строки 1220-1243: Изображения в основной галерее зала
            <div class="room-gallery-slide">
                <img src="photos/armaloft/armaloft1.jpg" alt="Arma Loft - фото 1">
            </div>
            ↑ Здесь меняйте пути к фотографиям (src="photos/НАЗВАНИЕ_ЗАЛА/фото.jpg")
            ↑ Меняйте alt текст для описания фотографий


 ID ЗАЛА ДЛЯ БРОНИРОВАНИЯ                                                    

Строка 1260: <div class="booking-form-container" data-hall-id="armaloft">
            ↑ Здесь меняйте ID зала (должен совпадать с ключом в booking.php):
               - "armaloft" для Армалофт
               - "merkuri" для Меркури
               - "samolet" для Самолет
               - "rufer" для Руфер
               - "pulka" для Пулька



 ХАРАКТЕРИСТИКИ ЗАЛА                                                         

Строки 1352-1369: Блок с характеристиками зала
            <div class="room-characteristic-item">
                <span>ДВУХУРОВНЕВЫЙ ЗАЛ. НИЖНИЙ УРОВЕНЬ - 120M², ВЕРХНИЙ 50M²</span>
            </div>
            <div class="room-characteristic-item">
                <span>ДО 120 ЧЕЛОВЕК</span>
            </div>
            ↑ Здесь меняйте описание характеристик зала (площадь, вместимость и т.д.)



 ОПИСАНИЕ ТОГО, ЧТО НАХОДИТСЯ В ЗАЛЕ                                         

Строки 1372-1422: Блок "Что находится в зале?"
            <div class="room-features-column">
                <div class="room-features-column-title">Внизу</div>
                <div class="room-feature-item">
                    <span>Танцпол с профессиональным светом, звуком и дымом</span>
                </div>
                ...
            </div>
            <div class="room-features-column">
                <div class="room-features-column-title">Вверху</div>
                <div class="room-feature-item">
                    <span>Столы для «ресторанной» рассадки гостей</span>
                </div>
                ...
            </div>
            ↑ Здесь меняйте описание того, что находится в зале
            ↑ Можно добавлять/удалять элементы <div class="room-feature-item">



 ПРИМЕЧАНИЕ                                                                   

Строки 1425-1429: Блок с примечанием
            <div class="room-note">
                <p class="room-note-text">
                    <strong>Примечание:</strong> Arma Loft – находится в нескольких минутах...
                </p>
            </div>
            ↑ Здесь меняйте текст примечания о зале



 ЗАГОЛОВОК СТРАНИЦЫ (В <head>)                                               

Строка 6: <title>Arma Loft - Зал Армалофт</title>
         ↑ Здесь меняйте заголовок страницы для SEO




2. КАК ЗАМЕНИТЬ ID УСЛУГИ (ПРИ БРОНИРОВАНИИ ЗАЛА) В /api/routes/booking.php


Файл: /api/routes/booking.php


 МАППИНГ ЗАЛОВ НА ID УСЛУГИ В YCLIENTS                                       

Строки 138-148: Массив $hallServiceMapping
            // Маппинг залов на service_id в YClients (ID услуги для бронирования)
            $hallServiceMapping = [
                'armaloft' => '25829928', // ID услуги для бронирования зала
                'mercury' => '25829928',
                'merkuri' => '25829928',
                'airplane' => '25829928',
                'samolet' => '25829928',
                'rufer' => '25829928',
                'pulka' => '25829928'
            ];
            ↑ Здесь меняйте ID услуги для каждого зала
            ↑ Формат: 'НАЗВАНИЕ_ЗАЛА' => 'ID_УСЛУГИ'
            ↑ Например: 'armaloft' => '12345678'



 ДЕФОЛТНЫЙ ID УСЛУГИ                                                         

Строка 151: $defaultServiceId = '25829928'; // ID услуги для бронирования зала
           ↑ Здесь меняйте дефолтный ID услуги (используется, если зал не найден в маппинге)



 ВАЖНО!                                                                      
- ID услуги должен быть строкой в кавычках (например: '25829928')
- Название зала в массиве должно совпадать с data-hall-id в HTML файле
- После изменения ID услуги перезапустите сервер (если используется)
- Проверьте, что новый ID услуги существует в вашей системе YClients
